## 菜单

使用耿多的设置来管理和组织菜单。菜单项可以根据分配的位置进行筛选。

### 菜单设置

下表显示了所有额外的菜单设置。

| 设置   | 说明 |
| ---------- | ------------------------------ |
| *列*   | 菜单下拉项中最多显示4列                                                                                                              |
| *宽度* | 输入像素值以设置下拉项的宽度                                                                                                         |
| *图标* | 从选择框中的图标预览中选择一个图标，将其分配给菜单项或者子项。也可以使用自定义图像。输入Joomla中完整路径，例如*/images/my-image.jpg* |
